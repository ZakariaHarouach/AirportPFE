this project was made with react and laravel for the education
and to run the project you should be already have composer for laravel and node.js
to run the backend : after opening the backend folder with CMD run thid command (composer install) after finishing run this command (php artisan serve)
to run the front end (the backend should be runing) : after opening the frontend folder with CMD run this command (npm install) after finishing run this command (npm run dev)
